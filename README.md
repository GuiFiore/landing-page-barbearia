# 💈 KingsCut — Landing Page

Landing page profissional desenvolvida para barbearia premium.

## 🔗 Demo ao vivo
[Ver projeto online](https://landing-page-barbearia-chi.vercel.app) 

## 🛠️ Tecnologias
- HTML5 semântico
- CSS3 (animações, Grid, Flexbox, responsivo)
- JavaScript vanilla (scroll reveal, cursor customizado)

## ✨ Funcionalidades
- Design premium dark com detalhes dourados
- Seções: Hero, Serviços, Barbeiros, Depoimentos e CTA
- Botão de agendamento direto pelo WhatsApp
- Animações de entrada ao rolar a página
- 100% responsivo para mobile
